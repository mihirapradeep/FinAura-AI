# FinAura-AI
FinAura AI is a financial wellness web app designed to help users understand, monitor, and improve their financial health in real time. FinAura AI is a multipage, interactive financial wellness web application built using HTML.

The app introduces a Financial Health Index (FHI) to measure a user’s financial stability based on income, categorized spending, and savings behavior combined with AI-style predictions and smart financial advice.
Designed with mobile-app style interface, FinAura transforms basic budgeting into an engaging and motivational experience.

Problem Statement
Many individuals:
	•	Do not track categorized expenses
	•	Overspend without awareness
	•	Lack a measurable financial health score
	•	Receive no intelligent financial guidance

Traditional expense trackers show data, but don’t interpret it meaningfully.

Our Solution
FinAura AI provides:
	•Financial Health Index (FHI) calculation
	•Income & categorized transaction tracking
	•Doughnut chart visualization
	•AI-style month-end balance prediction
	•Badge system based on financial discipline
	•Personalized AI advisor page
	
Financial Health Index (FHI)
The Financial Health Index is calculated using:

Savings Rate = (Savings ÷ Income) × 100
Spending Stability = 100 − (Total Spending ÷ Income × 100)

FHI Formula:
FHI = (Savings Rate × 0.4) + (Spending Stability × 0.3) + (50 × 0.3)
